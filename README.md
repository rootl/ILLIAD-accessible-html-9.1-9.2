# ILLIAD-accessible-html-9.1-9.2
This repo hosts WCAG 2.1/2.2 compliant ILLiad webpages for SUNY Geneseo Milne Library that can be shared with other institutions.

Please note: Many institutions have their own customizations to their ILLiad webpages.

Do a code compare of your institution's webpage file(s) with the file(s) in this repo.   Then, make the appropriate changes to your webpage files.  

DO NOT simply upload and replace your ILLiad webpage files with the ones from this repo as this could wipe out other customizations you have made to your webpages.
